# 🍽️ 나만의 맛집 사이트 만들기

📌 **사용자는 원하는 맛집을 확인하고, 관리자는 맛집 정보를 저장 및 수정할 수 있는 웹사이트입니다.**  
네이버 맵 API를 활용하여 위치 정보를 확인하고, 검색 기능을 통해 원하는 맛집을 손쉽게 찾을 수 있습니다.  

---
